# Twitter Clone

## 개요

- firebase, react를 사용한 실시간 SNS
- 글과 사진 업로드, 수정, 삭제 구현
- 사용자의 닉네임, 프로필 사진 수정 가능

## 사용한 기술
- Front-end : `react` `react-router-dom` `styled-components`
- Back-end : `firebase`
- Deploy : `gh-pages`
