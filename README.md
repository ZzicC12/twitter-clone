# firebase를 사용한 트위터 클론

## 개요
- [링크](https://zzicc12.github.io/twitter-clone/#/)
- firebase, react를 사용한 실시간 SNS
- 글과 사진 업로드, 수정, 삭제 구현
- 사용자의 닉네임, 프로필 사진 수정 가능

## 사용한 기술
- Front-end : `react` `react-router-dom` `styled-components`
- Back-end : `firebase`
- Deploy : `gh-pages`

## 프로젝트를 통해 배운 점
- firebase auth를 사용한 이메일로 로그인하는 법
- firebase storage를 사용한 사진 업로드하는 법
- firestore DB를 사용한 실시간 데이터 CRUD하는 법
- styled-components를 사용한 컴포넌트 스타일링 하는 법
